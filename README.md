# Diegetic Audio Control: In-World Volume Control for STALKER Anomaly

Independent volume sliders for radios, megaphones, guitars, and harmonicas. Hooks directly into the audio subsystems that play in-world sound. Each source gets its own volume, enable/disable toggle, and where applicable a pause multiplier. A master volume sits on top. All changes apply immediately through MCM.

[Releases](https://github.com/damiansirbu-stalker/Diegetic-Audio-Control/releases)

Requires: Anomaly 1.5.3, Modded exes, [xlibs](https://www.moddb.com/mods/stalker-anomaly/addons/xlibs-1001), MCM
